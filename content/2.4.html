Configuración de una Estación de Trabajo PLC

Sigue estos pasos para preparar tu hardware y software y empezar a programar con un PLC Allen-Bradley.

Paso 1: Preparación del Hardware

Antes de empezar con el software, asegúrate de que tu hardware esté listo. Una estación de trabajo de PLC modular típicamente consiste en los siguientes componentes:

  • **Chasis o Rack:** Es la estructura donde se montan todos los módulos. Asegúrate de que tenga el tamaño adecuado para tus módulos.
  • **Fuente de Alimentación:** Proporciona la energía necesaria a todos los módulos. Conéctala a una fuente de energía confiable y verifica que los indicadores LED estén encendidos.
  • **Módulo de CPU:** El cerebro del sistema. Insértalo en el slot designado (generalmente el primero a la izquierda).
  • **Módulos de E/S:** Inserta los módulos de entrada y salida que vayas a utilizar. Conecta los sensores y actuadores a las terminales correctas.

Es crucial asegurarse de que todos los módulos estén bien encajados en el chasis y que el cableado de alimentación esté correcto para evitar daños en el equipo.

Paso 2: Instalación del Software

Para programar un PLC Allen-Bradley, necesitas el software de desarrollo adecuado. Para los modelos modernos como ControlLogix o CompactLogix, el software principal es **Studio 5000 Logix Designer**.

  • **Verifica los requisitos del sistema:** Asegúrate de que tu computadora cumpla con los requisitos mínimos de hardware y software para la versión de Studio 5000 que vas a instalar.
  • **Instala el software:** Sigue las instrucciones del instalador. Ten en cuenta que este software puede requerir una licencia y que su instalación puede tomar un tiempo considerable.
  • **Instala el software de comunicaciones (FactoryTalk Linx):** Este es un componente crucial que permite a Studio 5000 comunicarse con el PLC a través de la red (Ethernet/IP, USB, etc.). A menudo se instala junto con Studio 5000.

Una vez instalado, abre el software para asegurarte de que se ejecuta correctamente antes de proceder a la conexión.

Paso 3: Conexión Física y de Red

Ahora es el momento de conectar tu computadora al PLC. La conexión más común para los PLC modernos es a través de Ethernet.

  • **Conexión directa:** Conecta un cable Ethernet directamente desde tu PC al puerto Ethernet de la CPU del PLC. Para esto, a veces se requiere un cable de cruce (crossover cable), aunque la mayoría de las tarjetas de red modernas lo detectan automáticamente.
  • **Conexión a través de un switch:** En un entorno industrial, es más común conectar el PLC a un switch industrial y tu PC al mismo switch.
  • **Conexión USB:** Algunos modelos de PLC Allen-Bradley también tienen un puerto USB. Puedes usar un cable USB estándar para una conexión temporal directa.

Asegúrate de que los indicadores LED de la tarjeta de red en el PLC muestren actividad para confirmar la conexión física.

Paso 4: Configuración Inicial en el Software

Finalmente, configura la comunicación entre tu software y el PLC.

  • **Configura el driver de comunicación:** En FactoryTalk Linx (o RSLinx Classic), crea un nuevo driver para el tipo de conexión que estés usando (por ejemplo, "Ethernet/IP" o "USB").
  • **Navega hasta el PLC:** Una vez que el driver esté activo, debería escanear la red y mostrar tu PLC en el árbol de navegación.
  • **Crea un proyecto en Studio 5000:** Inicia un nuevo proyecto y selecciona el modelo de CPU de tu PLC.
  • **Configura el árbol de E/S:** Dentro del proyecto, añade los módulos de E/S que tienes físicamente instalados en tu chasis.

Con estos pasos, tu estación de trabajo estará lista para comenzar con la programación del PLC.
